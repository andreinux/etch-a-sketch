# Etch-a-Sketch

A browser-based sketchpad built as part of The Odin Project Foundations curriculum.

## Overview

This project recreates the classic Etch-a-Sketch toy using HTML, CSS, and JavaScript. Users can draw by clicking and dragging across a grid of squares and generate new sketchpads of different sizes.

## Features

- 16×16 default sketchpad
- Adjustable grid size up to 100×100
- Click-and-drag drawing
- Dynamic grid generation using JavaScript
- Erase button to clear the sketchpad
- Responsive square sizing while maintaining the same sketchpad dimensions

## Built With

- JavaScript (Vanilla JS)
